About This Project

Landing Page Overview

This restaurant landing page, titled "Home of Flavors", is designed using Tailwind CSS, a utility-first framework for rapidly building custom user interfaces. The page is fully responsive and includes a hero section with headline and call-to-action, specialties menu, testimonials, and a footer. It's built to be elegant, functional, and adaptable across screen sizes.

The site includes multiple sections such as:

  • Navbar with working internal navigation
  • Hero image with slogan and buttons
  • Specialties grid (vegetarian options)
  • Customer testimonials
  • Contact information and social media links
